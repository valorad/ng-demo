组件样式

BUTTON 按钮

  1. DEFAULT 默认按钮

    传入参数：btnDef

  2. SUBMIT 提交按钮

    传入参数：btnSub

  3. Cancel 取消按钮

    传入参数：btnCancel

  4. LINK 扁平文字按钮

    传入参数：btnLink

FORM 表单

  1. INPUT 输入框

    1. DEFAULT 默认输入

      传入参数：iptDef

    2. WARN 警告

      传入参数：iptWarn

    3. ERROR 错误

      传入参数：iptErr

  2. SELECT 选择框

目前我们公共颜色样式表有如下所示：

  1. BgColor 背景颜色

    命名规范：bgColor加颜色编码，如背景为红色500，则为bgRed500

  2. Color 字体颜色

    命名规范：Color加颜色编码，如字体为红色500，则为red500

如果项目中需要其他公共样式，命名规范必须遵循以上规范：修饰符加颜色加代码！
